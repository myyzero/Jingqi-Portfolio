## 🛤️ 详细步骤

### 🔨 第1站: 本地测试

**目标:** 确保网站可以正常构建

**命令:**
```bash
npm run build
npx serve dist
```

**检查清单:**
- [ ] 构建成功无错误
- [ ] 网站可以在localhost访问
- [ ] 所有页面正常显示
- [ ] 图片和视频正常加载

**预计时间:** 5分钟

**如果成功:** 继续第2站 ✈️

---

### 📦 第2站: GitHub准备

**目标:** 将代码推送到GitHub

**步骤:**
1. 在 github.com/new 创建新仓库
2. 运行Git命令（见下方）
3. 刷新GitHub页面确认代码已上传

**命令:**
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/仓库名.git
git branch -M main
git push -u origin main
```

**检查清单:**
- [ ] GitHub仓库已创建
- [ ] 代码成功推送
- [ ] 在GitHub可以看到所有文件

**预计时间:** 5分钟

**如果成功:** 继续第3站 ✈️

---

### 🌟 第3站: Vercel部署

**目标:** 将网站部署上线

**步骤:**
1. 访问 vercel.com
2. 用GitHub登录
3. 导入你的仓库
4. 点击Deploy
5. 等待构建完成

**检查清单:**
- [ ] Vercel账号已创建
- [ ] 项目成功导入
- [ ] 构建成功
- [ ] 获得公开网址

**预计时间:** 5分钟

**如果成功:** 🎉 恭喜！网站上线！
